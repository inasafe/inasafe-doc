Toolbar¶

Here the different icons (buttons) in the toolbar and the function of each are explained.

Toggle InaSAFE Dock¶

This button shows and hides the InaSAFE dock. After enabling the dock you are able to move it around your screen. Dock it as a panel within the QGIS interface, or undock it so that it floats as a separate window.

Find more information about the dock in the InaSAFE Panel section.

InaSAFE Keyword Editor¶

The Keyword Editor button opens the editor which is is used to edit the keywords needed for InaSAFE to create useful output. Keywords are explained in more detail in Keywords System.

InaSAFE Options¶

This button opens the InaSAFE Options window which is described in detail in Options.

InaSAFE Impact Functions Browser¶

This button opens the Impact Function Browser, which allows you to filter and browse impact functions that are available in InaSAFE. More information can be found in Understand Impact Functions.

InaSAFE Minimum Needs Tool¶

This tool calculates minimum needs for evacuated people. For guidance on how to use this tool visit Minimum Needs Tool.

InaSAFE Converter¶

This tool converts an earthquake ‘shakemap’ that is in grid xml format to a GeoTIFF file. For more detailed information visit Shakemap Converter.

InaSAFE Batch Runner¶

The Batch Runner is used to load saved scenarios (Save Scenario) and batch run them in one go. Find more information about this tool in Batch Runner.

Save current scenario¶

This is the tool to prepare/save scenarios for the InaSAFE Batch Runner tool. It lets you save the current visible scenario in QGIS to a .txt file. Then this file can be opened as a scenario again in Batch runner and recalculated. A more detailed description is available in Batch Runner.

InaSAFE OpenStreetMap Downloader¶

This tool fetches building (structure) data from OSM.
